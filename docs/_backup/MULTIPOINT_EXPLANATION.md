# 多点模式说明

## 为什么只连接了2个点？

从日志可以看到：
```
[DEBUG] Stored points: 1 total
[DEBUG] Stored points: 2 total
[DEBUG] Right-click: Finishing connect_multi with 2 points
[DEBUG] Created multi-point connect marker CONNECT_0 with 2 points
```

**原因**: 你只点击了2个点就右键完成了，所以路径只连接这2个点。

## 多点模式的正确使用

### 要看到多点效果，需要点击3个或更多点：

```
左键点击 1 → 添加第1个点
左键点击 2 → 添加第2个点  
左键点击 3 → 添加第3个点  ← 这里继续点击
左键点击 4 → 添加第4个点  ← 可以继续添加更多
左键点击 5 → 添加第5个点
右键点击   → 完成，创建连接所有5个点的路径
```

### 2点 vs 多点的区别

**2 Points 模式** (传统模式):
- 点击2次自动完成
- 适合简单的两点连接

**Multi Points 模式** (多点模式):
- 可以点击任意多次（最少2次）
- 右键完成
- 适合复杂路径

## 测试建议

### 测试 3 点路径
1. 选择 "Multi Points"
2. 点击 "Connect" 按钮
3. 左键点击 3 个不同位置
4. 右键完成
5. 应该看到连接 3 个点的路径

### 测试 5 点路径
1. 选择 "Multi Points"
2. 点击 "Cut" 按钮
3. 左键点击 5 个位置
4. 右键完成
5. 应该看到连接 5 个点的路径

## 视觉效果

### 2点 CONNECT
```
  O========O
 起点      终点
```

### 5点 CONNECT
```
  O====o====o====o====O
 起点  中间点 中间点 中间点 终点
```

- O = 大圆圈（端点）
- o = 小圆圈（中间点）
- = = 连接线

## 常见误解

❌ **错误**: "我点了2个点，为什么没有看到多点效果？"
✓ **正确**: 多点效果需要3个或更多点才能体现

❌ **错误**: "多点模式应该自动添加更多点"
✓ **正确**: 每个点都需要手动点击，右键完成

❌ **错误**: "2点模式和多点模式有什么区别？"
✓ **正确**: 
  - 2点模式：点击2次自动完成
  - 多点模式：可以点击多次，右键完成

## 实际应用场景

### 使用 2 Points 模式
- 简单的两点切割
- 两点之间的连接
- 快速标记

### 使用 Multi Points 模式
- 复杂的切割路径（需要转弯）
- 多点电气连接
- 沿着复杂形状的路径
- 需要经过多个中间点的连接

## 示例

### 场景1: L形切割
```
需要切割一个L形路径：

  1---2
      |
      3

使用 Multi Points 模式:
1. 左键点击位置1
2. 左键点击位置2
3. 左键点击位置3
4. 右键完成
结果: 创建连接1-2-3的路径
```

### 场景2: 之字形连接
```
需要创建之字形连接：

  1   3   5
   \ / \ /
    2   4

使用 Multi Points 模式:
1. 左键点击位置1
2. 左键点击位置2
3. 左键点击位置3
4. 左键点击位置4
5. 左键点击位置5
6. 右键完成
结果: 创建连接1-2-3-4-5的之字形路径
```

## 总结

- **2点 = 简单直线**
- **3点+ = 复杂路径**
- **左键 = 添加点**
- **右键 = 完成**

多点模式的优势在于可以创建任意复杂的路径，但需要点击足够多的点才能体现出来！
