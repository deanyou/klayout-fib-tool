# 标尺颜色快速设置指南

## 设置为纯白色

### 步骤

1. 打开 KLayout
2. `KLayout → Preferences` (macOS) 或 `File → Setup` (Windows/Linux)
3. 点击 `Display` 标签
4. 找到 `Rulers/Annotations` 设置
5. 设置颜色为白色：RGB `(255, 255, 255)`
6. 点击 `OK`

### 效果

- ✅ 十字定位线：纯白色
- ✅ 尺寸标尺（ΔX, ΔY）：纯白色
- ✅ 比例尺：纯白色

### 验证

创建一个标尺工具，检查是否显示为白色。
